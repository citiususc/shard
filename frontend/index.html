SHARD

An interactive workbench that turns natural-language data constraints into ontology-grounded SHACL shapes. Work on one constraint at a time, or process a structured batch into shapes you can inspect, edit, validate and export.

GitHub repository

Edit, validate and approve the resulting shapes before exporting Turtle.

Authoring workflows

Start with one constraint or process a complete batch.

RuleShape

Resolve one natural-language data constraint against an ontology, then generate a SHACL shape that you can inspect, edit, validate and accept.

BatchShapes

Upload a structured batch of data constraints. SHARD resolves each constraint and streams the generated shapes into a review queue before export.
