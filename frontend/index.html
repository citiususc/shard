Workflow 1

RuleShape

Write a data constraint, pick the ontology target, and generate a single SHACL shape you can edit, validate and accept.

  • Semantic ontology-term ranking
  • Target-aware generation context
  • Fast review before accepting

Workflow 2

BatchShapes

Upload a structured batch of data constraints (.html or .md). The pipeline resolves each extracted rule and streams generated shapes for review.

  • Batch validation and rule indexing
  • Shape queue with validation status
  • Export accepted Turtle shapes
